# `multiply()`

multiplies an amount from the money.

## Methods

### `multiply(float $number)`
**Parameters**:
1. `float $number` - a number on which the money will be multiplied.

**Returns**: `Money`

## Usage

```php
$money = money(1000);   // "$ 100"
$money->multiply(1.5);  // "$ 150"
```
