# `absolute()`

turns the amount of the monetary object into an absolute value. `$ -10.25 -> $ 10.25`

## Methods

### `absolute()`
**Returns**: `Money`

## Usage

```php
$money = money('-102500');  // $ -10.3
$money->absolute();         // $ 10.3
```
