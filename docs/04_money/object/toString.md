# `toString()`
represents Money object as a string.
You may not use it if you explicitly assign the object to a string.

All the ways to output money are mentioned [here](/docs/01_usage/output.md).

## Methods

### `toString()`
**Returns**: `string`

## Usage

```php
$money = money('1234000');
$money->toString(); // "$ 123.4"
```
